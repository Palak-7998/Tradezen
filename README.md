# TradeZen – MERN Stack Stock Trading Application

TradeZen is a full-stack stock trading web application developed using the MERN stack.
The project is designed and implemented to simulate core functionalities of a modern
trading platform while focusing on backend development and frontend integration.

This project was independently developed to gain hands-on experience in full-stack
web development and real-world application building.

---

## 🚀 Features
- Interactive dashboard to view holdings and positions
- Watchlist and order-related components
- RESTful backend APIs built using Express and Node.js
- MongoDB database integration
- JSON-based request and response handling
- Backend test route to verify server functionality
- Modular and reusable frontend components

---

## 🛠️ Tech Stack
- **Frontend:** React.js
- **Backend:** Node.js, Express.js
- **Database:** MongoDB
- **Language:** JavaScript
- **Tools:** Git, GitHub, VS Code

---

## 🎯 Project Objective
The objective of this project is to:
- Understand MERN stack application architecture
- Build and test backend APIs
- Work with MongoDB for data persistence
- Integrate frontend and backend effectively
- Strengthen full-stack development skills

---

## ▶️ How to Run the Project
1. Clone the repository
2. Install dependencies in both backend and frontend directories
3. Configure environment variables
4. Start backend and frontend servers
