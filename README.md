Zerodha Platform – Full Stack Web Application
This project is a fully functional full-stack web application inspired by Zerodha, built to demonstrate real-world frontend and backend integration. It includes user navigation, multiple product pages, routing, and backend connectivity for future scalability.
The application is designed to closely resemble a production-level platform with clean UI, reusable components, and a scalable architecture.
Features
Responsive and modern UI inspired by Zerodha
Multiple pages with client-side routing (Home, Products, Pricing, Support, Signup, etc.)
Reusable React components for clean code structure
Proper navigation using React Router
Frontend and backend structured for real deployment
Ready for backend APIs such as authentication, signup, and data handling
Production-ready build and deployment support
Tech Stack
Frontend
React.js
React Router DOM
HTML5, CSS3, JavaScript
Bootstrap
Backend
Node.js
Express.js (API-ready structure)
Tools & Deployment
Git & GitHub
Vercel (Frontend Deployment)
Render / Railway (Backend Deployment – supported)
How to Run the Project Locally
Clone the repository
Install dependencies
Start the frontend server
Start the backend server
Open the application in the browser
The application will run smoothly with routing and page navigation enabled.
Deployment
The frontend is deployed using Vercel
Backend can be deployed separately using Render / Railway
The project supports production builds and environment variables
